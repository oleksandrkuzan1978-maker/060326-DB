## Создание SQL таблицы с помощью DataFrame (pandas + SQLite)

```python
import pandas as pd
import sqlite3
```

### Что такое `pandas`?:

* `pandas` — библиотека для работы с табличными данными (DataFrame)
* `sqlite3` — встроенный модуль Python для работы с SQLite
* В отличии от `sqlite3`, `pandas` является внешней библиотекой. 
  * Поэтому, требует предварительной инсталляции:`pip install pandas`
* Вместе они позволяют легко:
  * создавать таблицы из DataFrame
  * сохранять данные в базу данных
  * читать данные обратно в DataFrame

---

### Создание DataFrame

```python
# Создаём словарь с данными
data = {
    'column1': ['value1', 'value2', 'value3'],
    'column2': ['value1', 'value2', 'value3'],
    'column3': ['value1', 'value2', 'value3'],
}

# Преобразуем словарь в DataFrame
df = pd.DataFrame(data)

# Выводим таблицу
print(df)
```

### Что такое `DataFrame`?

* `DataFrame` — это таблица (как в Excel)
* ключи словаря → названия колонок
* списки → строки значений
* все столбцы должны быть одинаковой длины

---

### Загрузка DataFrame в SQLite

#### 1. Создаём подключение к СУБД SQLite

```python
# Подключение к базе данных (файл создаётся автоматически, если его нет)
conn = sqlite3.connect('students.db')
```

#### 2. `pandas` содержит встроенный метод `to_sql`, благодаря чему

* без использования команды `CREATE TABLE` создаётся файл базы `students.db`
  * если файл уже есть — просто открывается
  * это “точка входа” в базу данных

---

```python
# Записываем DataFrame в SQLite как таблицу
df.to_sql(
    'new_table',   # имя таблицы в БД
    conn,          # подключение к базе
    if_exists='replace',  # заменить таблицу, если она уже существует
    index=False    # не сохранять индекс pandas как отдельный столбец
)
```

#### 3. Краткие пояснения параметров метода:

* `to_sql()` автоматически создаёт таблицу
* `if_exists='replace'`:
  * удаляет старую таблицу и создаёт новую
* альтернативы:
  * `'fail'` — ошибка если таблица существует
  * `'append'` — добавить данные в существующую таблицу

---

### Проверка содержимого таблицы

Для чтения данных из таблицы `pandas` имеет метод `.read_sql_query`

```python
# Выполняем SQL-запрос и сразу получаем DataFrame
df_from_db = pd.read_sql_query(
    'SELECT * FROM new_table',
    conn
)

# Вывод результата
print(df_from_db)
```

---

#### Таким образом `read_sql_query()`:

* выполняет SQL-запрос
* возвращает результат сразу в DataFrame
* и это самый удобный способ интеграции pandas + SQLite
